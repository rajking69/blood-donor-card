# 🩸 Blood Donor Card Generator

A clean, fast Blood Donor Card Generator built with Next.js 14, React 18, and Tailwind CSS.
Fill in a few fields, upload a photo, and instantly get a premium-quality donor card — downloadable as PNG or PDF.

---

## ✨ Features

- **Live preview** — card updates as you type
- **Portrait card** (9:16 ratio) with red medical aesthetic
- **QR code** auto-generated from name, phone & email
- **PNG export** at 3× pixel ratio for print quality
- **PDF export** centered on A4 page
- Responsive — works on mobile & desktop

---

## 🚀 Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### 3. Build for production

```bash
npm run build
npm start
```

## 🎨 Design

- **Primary color:** `#D90429`
- **Font:** Poppins (headings) + Inter (body)
- **Card ratio:** Portrait 9:16
- **Corner radius:** 24px
- **Shadow:** `0 20px 60px rgba(217,4,41,0.18)`
